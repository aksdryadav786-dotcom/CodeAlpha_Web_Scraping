# 🚀 CodeAlpha Web Scraping Project

## 📌 Project Overview
This project focuses on web scraping quotes data from a website using Python. The data is extracted from the website and stored in a CSV file for analysis and future use.

## 🛠 Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
- Google Colab

## 🔍 Features
- Extract quotes from website
- Extract author names
- Store scraped data in CSV file
- Data cleaning using Pandas

## 📂 Files Included
- `CodeAlpha_Web_Scraping.ipynb` → Main project notebook
- `quotes_data.csv` → Extracted quotes dataset

## 📈 Sample Output
- Quote Text
- Author Name
- CSV Data Export
